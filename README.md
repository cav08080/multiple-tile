# かけ算フラッシュカード

特別支援教育向けにユニバーサルデザイン（UD）原則に基づいて設計された、かけ算練習用フラッシュカードアプリケーションです。

## 🎯 特徴

### ユニバーサルデザイン（UD）対応
- **視覚的な安定性**: 9×9の固定グリッド表示で、視覚的な予測可能性を提供
- **左下起点**: 問題は左下（1×1）から積み上がるように表示
- **縦線のみ**: 白いブロック内部は縦線のみで、横線を排除（列の数を数えやすく）
- **完璧な枠線**: オーバーレイ技術により、ズレや二重線のない正確な表示

### シンプルな操作
- **ボタン不要**: カードクリックまたはスペースキーのみで操作
- **直感的**: 問題→答え→次の問題へとスムーズに進行
- **キーボード対応**: スペースキーまたはEnterキーで進む

### 学習機能
- **全81問**: 1×1から9×9までの全問題を網羅
- **ランダム出題**: 毎回異なる順序で出題
- **自動繰り返し**: 全問終了後、自動的にシャッフルして再開

## 🎮 使い方

1. **問題を見る**: グリッドの形から答えを考えます
   - 縦のマス数 = 被乗数（かけられる数）
   - 横のマス数 = 乗数（かける数）
   - 右上の数字 = 答え

2. **答えを確認**: カードをクリック（またはスペースキー）して式を表示

3. **次の問題へ**: もう一度クリック（またはスペースキー）

## 🛠️ 技術仕様

- **HTML5**: セマンティックなマークアップ
- **CSS3**: モダンなレイアウト（CSS Grid、Flexbox）
- **Vanilla JavaScript**: フレームワーク不要の軽量実装
- **レスポンシブ**: 様々なデバイスサイズに対応
